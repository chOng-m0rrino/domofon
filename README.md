# Home Assistant Community Add-on: Умный Дом.ru

## ✅ Форк с исправленным api.


## Что это?

Это дополнение (Add-on) для Home Assistant, которое позволяет интегрировать умный домофон в систему Home Assistant

## Установка через Магазин приложений

Для установки необоходимо в зайти в Home Assistant -> Supervisor -> Add-on Store -> Menu -> Repositories

В появившемся окне необходимо добавить: `https://github.com/chOng-m0rrino/domofon` и нажать `Add`

После установки необходимо зайти в Web UI (Open WEB UI) и авторизоваться с помощью телефона и смс.

## Установка локальная

Нажать большую зеленую кнопку "Code" -> Download Zip . Скачиваем архив. Открываем или Распаковываем.

Для установки необоходимо перейти в папку \addons (например \\ha.local\addons) и скопировать содержимое папки domru из архива сюда.

После в "Магазин приложений", проверить налицие обновлений, появится категория Local apps в ней "Домофон", устанавливаем.

После установки необходимо зайти в Web UI (Open WEB UI) и авторизоваться с помощью телефона и смс.
